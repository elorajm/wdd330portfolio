# WDD330 Portfolio Site

A clean, single-page portfolio submission site with all required project links.

## Includes
- Reflection video (under 3 minutes)
- Personal final project links (Render + GitHub)
- SleepOutside links (GitHub + Netlify + Trello)

## Run locally
Open `index.html` in your browser.

## Deploy to Render
1. Push this folder to a GitHub repository.
2. In Render, create a new **Static Site** from that repository.
3. Render will detect `render.yaml` automatically.
4. If prompted manually, set:
   - Build Command: *(leave empty)*
   - Publish Directory: `.`

After deploy, your portfolio URL will be live on Render.
